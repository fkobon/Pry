Ajout de validateur

Pour créer des validateurs supplémentaires il suffit d'étendre Validate_Abstract.
Actuellement neuf validateurs sont disponibles :

  • Alnum : Validation de caractère alphanumérique
  • Alpha : Validateur de chaine de caractères (inclus espace et -)
  • CP : Validation de code postaux français (et corse)
  • Date : Validation de date au format fr/us/sql/sql datetime
  • Digit : Validation de nombre
  • Email : Validation d'adresse email. Equivalent de Util_String::isMail()
  • Equal : Validation d'égalité
  • Interval : Vérifie si une valeur numérique est comprise dans un interval
  • Time : Validation d'une heure au format HH:ii:ss
