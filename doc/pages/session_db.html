Présentation

La classe DbStorage permet d'utiliser les session avec une base de données plutôt que de passer par le système classique de PHP.
La table gérant les sessions devra contenir au minimum 3 colonnes :

  • Une colonne pour un id
  • Une colonne pour les données
  • Une colonne pour un temps (datetime)
