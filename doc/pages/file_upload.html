Présentation

La classe File_Upload permet l'envoi de un ou plusieurs fichier de manière sécurisé.
Par défaut la classe accepte les extensions jpg,png,gif,zip,rar,avi,wmv,mpg,pdf,doc,docx,xls,txt et les mimes image/gif,image/jpeg,image/png,text/plain,application/pdf.
Le troisième paramètre optionnel du constructeur permet de définir comment vérifier les types mimes. Par défaut le type mime envoyé par HTTP est utilisé , il est conseillé d'utiliser MIME_CHECK_FINIFO si l'extension finfo est disponible.
