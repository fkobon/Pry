Présentation

La classe abstraite BaseController permet d'ajouter des fonctionnalité utile à tous les controller d'une application.
Dans le cas ou l'application n'utiliserait pas Twig les propriété $view et $template pourrait être vide , idem pour $db.

Création d'une classe héritant de BaseController. L'utilisation du paramètre de lang est optionnel dans le cas d'un site non multilangue
