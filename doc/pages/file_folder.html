Présentation

La classe File_FolderManager permet de gérer les dossiers et les opérations associées.

Exemple

Lister récursivement un dossier

Lister récursivement un dossier mais en appliquant des filtres sur les types de fichier

Zipper tous les fichier d'un dossier (non récursif)
Si des filtres sont définis ils s'appliqueront.

Consulter l'API pour voir toutes les possibilité de la classe
